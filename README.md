# ✊✋✌️ Rock Paper Scissors Game

A fun and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript.
This project features responsive design, score tracking with localStorage, emoji-style game icons, and a colorful modern UI for an engaging gameplay experience.

# 🚀 Features
🎮 Play Rock, Paper, Scissors against the computer

🤖 Random computer move generator

📊 Live score tracking

💾 Score saved using localStorage

🔄 Reset score button

📱 Fully responsive design for mobile, tablet, and desktop

🎨 Colorful and attractive UI

🖼️ Emoji-based game visuals

# 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)

# 🎯 How It Works
Click on Rock, Paper, or Scissors. 

The computer randomly selects a move.

The game compares both moves and displays:

Win
Lose
Tie

Scores are automatically updated and stored in browser storage.

# 📱 Responsive Design

The game includes multiple media queries for:

Mobile devices
Tablets
Laptops
Large desktop screens

This ensures smooth gameplay on all screen sizes.
